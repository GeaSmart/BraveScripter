BRAVE SCRIPTER
==============

Aplicación que basado en un archivo excel y una(s) tabla(s) de una base de datos genera scripts para insertar dichos datos.

Primeros pasos:

1. Establecer la configuración en el archivo config.json
2. Crear la carpeta definida en 'ExcelPath' y 'OutputSqlPath'
3. Su archivo excel deberá contener los datos para insertar
4. Ese archivo deberá tener tantas hojas de cálculo como tablas configuradas en 'TablasDestino'
5. El nombre de la hoja deberá coincidir con el nombre de su tabla
6. En la primera fila de cada hoja van los nombres de las columnas
7. Ejecute el archivo BraveScripter.exe
